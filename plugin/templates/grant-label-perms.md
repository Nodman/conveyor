### Board-write grant (written by scaffold --grant-label-perms)

Conveyor board lifecycle writes to this project's own GitHub repo are part
of the standing workflow: gh issue create/comment/edit, gh pr
create/comment/edit, label add/remove, board card moves via gh project or
card.sh, pushing feature branches, and filing backlog issues for
out-of-scope findings discovered during a task.
Merging PRs and moving cards to Done remain human-only and are NOT covered.
